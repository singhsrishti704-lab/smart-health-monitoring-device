# Smart Health Monitoring Device

This mini project is a wearable device that monitors health parameters such as:
- Heart rate
- Body temperature
- Blood oxygen levels

## Features
- Real-time monitoring
- Alerts for abnormal readings
- Data visualization (optional)

## Tech Used
- Arduino / ESP32
- Sensors: Pulse, Temperature, SpO2
- C++ / Embedded C

## How to Run
1. Connect sensors to microcontroller
2. Upload the code via Arduino IDE
3. Monitor the output via serial monitor or app
